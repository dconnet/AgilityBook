Trial Context Menu

This context menu displays a small set of the main menu items can apply specifically to the item you just clicked on with your right mouse button.

  • Properties...
  • New

    • Dog...

    • Title...
    • Trial...
    • Run...
  • Duplicate
  • Cut
  • Copy
  • Paste
  • Delete
  • Reorder... - Displays a dialog allowing you to reorder the runs in the trial. Only runs on a given day can be reordered. If you move a run too high or too low, it will automatically sort back into the proper location.
  • Expand - Expand the selected item. Clicking on the + sign or using the right arrow key will also expand the item.
  • Collapse - Collapse the selected item. Clicking on the - sign or using the left arrow key will also expand the item.
