About Dialog

This dialog provides quick information about the program.

When discussing the program with others, it is often helpful to know which version you are discussing. This can be easily obtained in the first line.

The second paragraph provides information about the distribution rights. As noted, this program is free. However, we simply request that you make a donation to a local shelter or rescue group. This donation may in the form of volunteering also. We rely on your honesty for this.

In addition, there are a couple of hotlinks to provide easy access to some convenient email addresses and web sites.

The Check for Updates... button provides an easy way to see if there is a new version of the program available. See the Checking for updates for more information.
