Agility Record Book is a handy program for tracking all your agility records. And it's free! (Ok, we do ask that you make a contribution to your local animal shelter or rescue group.) It started life as a humble Excel spreadsheet but quickly grew too complicated with all the differences between the various venues. On December 28, 2002, Agility Record Book was born!

Agility Record Book is designed to be highly configurable. If a venue changes rules or a new venue comes along, there is no need to release a new version of the program! All we need to do is update the configuration which lives in your data file.
