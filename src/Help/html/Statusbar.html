Status bar

The Status bar provides some convenient information.

The left side of the status bar is used for displaying various status messages. Usually, it just says "Ready". Whenever you highlight a menu item, a description of that item will appear in the this area.

The right side of the status bar is broken into four pieces:

  • Number of items
    • Runs: Total number of runs and number of Qs.
    • Points: Not used.
    • Calendar: Number of visible calendar entries.
    • Training Log: Number of visible training log entries.
  • Filtered - Specifies it filtering is enabled on the current view.
  • CAP - This specifies whether CapLock is on.
  • NUM - This specified whether NumLock is on.
