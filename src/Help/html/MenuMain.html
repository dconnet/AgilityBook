Program Menu

The following items are available in the main program menu.

File Menu

  • New - Create a new file.
  • Open... - Open an existing file.
  • Save - Save the current file. If this is a new file, this performs the Save As... action.
  • Save As... - Prompts you for the location and name of a file and then saves it.
  • Import and Export... - Opens the Import Export Wizard dialog.
  • Find all linked files... - Find and display all the files currently linked to runs.
  • Print - Prints the current view.
  • Print Preview - Preview the current view for printing. The preview also allows you to print from there.
  • Print Setup... - Configuration your printer. This displays the standard Windows print configuration dialog.
  • Recent Files list - A list of files that you have recently opened. Simply select one of these files to open it.
  • Exit - Closes the Agility Record Book application. If your file has been changed, you will be prompted to save those changes.
Edit Menu
  • Undo - Undo the last action.
  • Duplicate - Duplicate the current selection.
  • Cut - Copy the current selection to the clipboard and then delete it.
  • Copy - Copy the current selection to the clipboard.
  • Paste - Paste the clipboard contents.
  • Select All - Select all items.
  • Find... - Display the Find dialog.
  • Find Next - Find the next item matching the last search criteria. If no search has been performed, the Find dialog is displayed.
  • Find Previous - Find the previous item matching the last search criteria. If no search has been performed, the Find dialog is displayed.
  • Configuration - Display the Configuration dialog.
Agility Menu
  • Dog
    • Properties... - Display the properties of the selected dog.
    • New... - Create a new dog.
    • Delete - Delete the selected dog and all the titles, trials and runs associated with it.
  • Title
    • Properties... - Display the properties of the current title.
    • New... - Create a new title for the current dog.
    • Delete - Delete the selected title.
  • Trial
    • Properties... - Display the properties for the selected trial.
    • New... - Create a new trial for the current dog.
    • Delete - Delete the selected trial and all the runs associated with it.
  • Run
    • Properties... - Display the properties for the selected run.
    • New... - Create a new run for the current trial.
    • Delete - Delete the selected run.
  • Calendar
    • Properties... - Display the properties for the selected calendar entry.
    • New... - Create a new calendar entry.
    • Delete - Delete the selected calendar entry.
    • Export... - Export the selected calendar entries.
    • Create Trial Entry... - Create a trial directly from the calendar entry.
  • Training
    • Properties... - Display the properties for the selected training log.
    • New... - Create a new training log.
    • Delete - Delete the selected training log.
Notes Menu
  • Clubs - Display the Club's Information dialog.
  • Judges - Display the Judge's Information dialog.
  • Locations - Display the Location Information dialog.
  • Search Notes... - Display the Find dialog and search each of the above items.
View Menu
  • Options... - This dialog displays the various options that can be set. The page that is initially shown is based on the view that is currently active.
  • Customize... - Customize the current view. This will display a different list of items that can be customized for each window in the four main tabs. It allows you to specify the fields you would like to see, and their order.
  • Show Newest Trials First - Show the newest trials first. This only affects the left side of the Runs view.
  • Show Runs By Trial - When this option is set, the right side of the Runs view shows only runs associated with the item selected in the left side. If you select a dog, all the runs will be visible. When you select a trial, only those runs in that trial will be visible. Unsetting this option displays all the runs for the current dog.
  • Show Hidden Titles - Show any titles that have been hidden. This option is also available in the Titles page.
  • Sort Points View... - Specify how to sort events in the Points View.
  • Include Table in YPS - Not including the table will subtract 5 seconds from your course time when computing your Yards Per Second.
  • Toolbar - Toggle the visibility of the tool bar.
  • Status Bar - Toggle the visibility of the status bar.
Window Menu
  • Next Pane - Navigate between two windows within the same view.
  • Previous Pane - Navigate between two windows within the same view.
Help Menu
  • Contents - Display the help contents.
  • Index - Display the help index.
  • Check for Updates... - Check to see if there are any updates (program or configuration) available. See Checking for updates for more information.
  • System Information... - Display information about the system.
  • Show Splash Screen... - Show the initial splash screen. Click anywhere to dismiss it. You can change the image using the Options dialog.
  • About Agility Record Book... - Display the About dialog.
