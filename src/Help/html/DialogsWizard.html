Import Export Wizard

The Import Export Wizard allows you to add data in and copy it out for use in other programs. For instance, the data analysis aspects of Agility Record Book are rather limited. This is by-design. There are many other programs in existence that can analyze data far better than I can. So this provides the facility to get the data out of this program in a form the other program can use.

The Wizard starts by determining what you want to do. Based on the selections you make in this dialog, the action is either completed, or you are taken to the import or export portion of the wizard.
