Import / Export Wizard

The Import Export Wizard allows you to import or export data from/to either a spreadsheet or another Agility Record Book file. The options available between the spreadsheet and Agility Record Book formats are slightly different, as you can see from the dialogs.

When you select in entry in the list, a description will be displayed on the right describing what will happen. When exporting for a spreadsheet, you need to specify the format. Also, when you import from a spreadsheet, you also need to specify the format.

When importing and exporting in the Agility Record Book format, the only additional information you need to supply is the name of the file.
