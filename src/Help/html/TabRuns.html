Runs

The Runs tab is made up of two main windows. The view on the left is known as the Tree, the one on the right is the Runs list.

When you select a run in the tree, the run will also be highlighted in the right side. The view that currently has focus will have the selected item drawn more visibly (in the above picture, the left view has focus).

The icons in the Tree represent different things. For a Dog, the icon is simply a dog's head . For a Run, the icon is a couple of agility obstacles . For a trial, the first icon (which will be either blank or a check mark) shows whether the trial's data has been verified. The second icon represents the venue. If the venue is not a pre-configured venue, the icon used will be the same as the toolbar icon for creating a new trial .

If you right-click on an item (or press the "Window Menu" key on the keyboard), one of 3 (dog, trial, run) different context menus will appear allowing you to perform a number of actions. (In the 'Runs' list, the run menu will appear.)

Also, double-clicking on an item (in either window), will bring up one of 3 (dog, trial, run) different dialogs (the Properties item in the context menu will do the same thing).
