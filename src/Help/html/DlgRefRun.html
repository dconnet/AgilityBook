Reference Run Dialog

The Reference Run dialog allows you to create or edit a run of another dog in this event.

Place is for where this dog placed.

Q? is the qualifying status.

Time is the time of the run.

Score is the score the dog earned. This will initially be set to the value of a clean run. This varies by venue.

Height is the dog's height. This is useful when comparing against dogs you normally compete against in other venues. Or, in venues that have multiple levels run the same course (like AKC Excellent A/B, CPE Levels 4,5,C). I will often record additional level information like "L4-16" for another dog if I were in "L5".
