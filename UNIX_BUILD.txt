Building on Ubuntu Linux:
Targeting GTK2 on Ubuntu

Needs (note, some aren't strictly needed):
sudo apt install git python3 python3-pip curl
sudo apt install libpango1.0-dev (still needed?)
sudo apt install libboost-all-dev libgtk2.0.dev (libgtk-3-dev)
sudo apt install libcanberra-gtk-module

Setup:
1. Download wx source (3.1.2) (extract to ~/devtools/wx/wxWidgets-3.1.2)
2. cd into directory where you want to clone
   git clone https://github.com/dconnet/AgilityBook.git
3. cd AgilityBook/build
4. Use build/setupwx.sh script to setup WX
5. cd ~/devtools/wx/wxWidgets-3.1.2/build-release
   make
6. cd .../AgilityBook
7. . build/setwxpath.sh 3.1 release
8. autoconf
9. mkdir bld
10. cd bld
11. ../configure
12. make
